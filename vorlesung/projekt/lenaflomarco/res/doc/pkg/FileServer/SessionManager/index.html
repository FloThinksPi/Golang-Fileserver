    Constants

    const (
    SESSION_KEY_LENGTH = 64

    SESSION_KEY_LENGTH Length of the SessionKey which gets used after Authentication for access control.

    func NewSession

    func NewSession(record SessionRecord) (err error)

    NewSession Checks if a supplied SessionRecord is completely filled, reqeust a random SessionID and adds this to the SessionRecord. The SessionRecord is then saved in the packageGlobal managersSessionStorage.

    func ValidateSession

    func ValidateSession(session string) (valid bool, err error)

    ValidateSession checks if a supplied SessionID exists in managersSessionStorage and returns true if the Session didnt time out.

    type SessionMap

    type SessionMap map[string]SessionRecord

    SessionMap arranges SessionRecords in a Map for faster Access by SessionID

    type SessionRecord

    type SessionRecord struct {
    Email       string    //Email
    Session     string    //SessionCookie
    SessionLast time.Time //Timestamp when last Interaction took place -> for session timeout
}

    SessionRecord Entry of a single user. Email is unique and identifies a user. Session is the sessionKey which is used for further authentication. SessionLast is a timestamp of the last access to the SessionRecord, this is used to implement a timeout functionality.

    type SessionStorage

    type SessionStorage struct {
    SessionMap
    sync.RWMutex
}

    SessionStorage RWmutex for concurrent access and prevention of Mutual Exclusion. Holds A SessionMap and adds Mutex functionality
